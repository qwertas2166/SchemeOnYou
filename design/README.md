# SchemeOnYou design workspace

Назначение: UI/UX/shortcut layout для построения схем БД в keyboard-first desktop editor.

Файлы:
- `requirements-summary.md` — сжатый контекст требований, на котором основан дизайн.
- `db-diagram-ui-ux-v01.md` — первая версия решения.
- `revision-log.md` — журнал итераций и критерии остановки.

Правило ревизий: каждая новая версия должна улучшать UX или явно фиксировать, что дальнейшие идеи ухудшают продукт.
